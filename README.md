# koswika-portfolio

My personal portfolio – built with Next.js, Tailwind CSS, and Bootstrap.  
It shows who I am, what I build, and how to reach me.

## What's inside

- Next.js (App Router, TypeScript)
- Tailwind CSS + Bootstrap 5
- FontAwesome & Devicon icons
- Dark theme
- GitHub repos fetched live

## Running locally

```bash
git clone https://github.com/koswika/koswika-portfolio
cd koswika-portfolio
npm install
npm run dev
